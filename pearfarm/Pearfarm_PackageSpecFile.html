Pearfarm_PackageSpecFile

A "file" object within the pearfarm spec.

Files are generally created for you automatically by the higher-level functions, but you can create them yourself if you like and then add the file with addFile().

The more typical use case is to modify an existing file by calling $spec->getFilePath().
